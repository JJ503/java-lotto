## 미션 흐름에 따른 문제 이해

1. 로또 구매 금액 입력하기
    1. 입력 값 유효성 검사
        - 값이 있어야 한다
        - 숫자여야 한다
        - 1,000원 단위여야 한다
        - 0원을 초과해야 한다
        - 위 조건에 부합하지 않는다면 예외 발생

2. 로또 구매
    1. 무작위 값의 로또 생성하기
        - 모든 값은 1 ~ 45 사이의 값을 무작위로 선택한다
        - 하나의 로또의 숫자 개수는 6개여야 한다
        - 하나의 로또에 중복되는 값이 있어선 안 된다
        - 값은 오름차순으로 정렬한다

3. 당첨 번호 입력하기
    1. 입력 값 유효성 검사
        - 값이 있어야 한다
        - 쉼표로 구분된 값은 6개가 있어야 한다
        - 모든 값은 숫자여야 한다
        - 모든 값은 1 ~ 45 사이여야 한다
        - 로또의 숫자들은 서로 중복되어선 안 된다
        - 위 조건에 부합하지 않는다면 예외 발생

4. 보너스 번호 입력하기
    1. 입력 값 유효성 검사
        - 값이 있어야 한다
        - 숫자여야 한다
        - 1 ~ 45 사이의 값이여야 한다
        - 입력한 당첨 번호와 중복되지 않는 값이어야 한다
        - 위 조건에 부합하지 않는다면 에외 발생

5. 당첨 통계 출력하기
    1. 일치 개수별 로또 개수 출력
        - 3 ~ 6개 일치까지 몇 개의 로또가 포함되는 지 출력한다
    2. 총 수익률을 출력한다
        - 수식 : (총 수익) / (구입금액) * 100
        - 수익률은 소수점 둘째 자리에서 반올림한다
        - 1000단위 별로 쉼표를 찍어주어야 한다
        - 예시

            ```plaintext
            ex. 100.0%, 51.5%, 1,000,000.0%
            총 수익률은 62.5%입니다.
            ```
